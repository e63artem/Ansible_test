LEMP стек, который разворачивается в контейнерах при помощи docker-compose файла, 
который в свою очередь запускается при помощи ansible playbook

Как сделал я:
- использовал vagrant, т.к. хотел показать навык применения
(wsl, vmware, virtualbox решил не использовать в данной задаче)

- стек разворачивается одной командой vagrant up в папке с vagrant file
(часть функционала можно перенести в ansible tasks для оптимизации и удобства (сделал не очень оптимизировано, как позволило время) )

- vagrant поднимает 2 машины, одна в качестве controlnode, другая имитирует server (поэтому можно легко масштабировать)

- на controlnode устанавливается ansible, запускает ansible playbook на машине server

- playbook устанавливает элементы для запуска docker-compose, запускает docker-compose.yml

- docker-compose.yml билдит образ для php, устанавливает образы для других сервисов 

- LEMP стек развернут
